# CAS Jalali - Mail & Chatter Bridge

Release 19.0.1.0.0 covers:

- Old/new values of tracked Date fields.
- Old/new values of tracked Datetime fields.
- Message date separators.
- Message timestamps for today, yesterday and older messages.
- Full Jalali date/time tooltips.
- Scheduled-message date/time labels.

The database remains Gregorian/UTC. Formatting is applied only to the payload
and browser presentation.
