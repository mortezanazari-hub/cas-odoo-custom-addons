# CAS Attendance Operations

ورودی عملیاتی آفلاین برای رخدادهای حضور:

- ثبت سریع چند نفر در یک نوبت توسط نگهبان
- ورود Excel/CSV دستگاه، فایل‌های جفت‌شده و دفتر Excel نگهبانی
- نگاشت شناسه دستگاه یا نام شیت به کارمند Odoo
- مرحله بازبینی برای شناسه ناشناس، رکورد میانی و خطا
- جلوگیری از ورود تکراری و حفظ فایل و بچ برای ممیزی
