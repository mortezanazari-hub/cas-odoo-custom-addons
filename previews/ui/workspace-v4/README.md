# CAS UI Preview — Workspace v4

| مشخصه | مقدار |
|---|---|
| وضعیت | `Non-Final Preview` |
| نسخه نمونه | `v4` |
| صفحه تغییرکرده | میزکار کاربر عادی |
| مبنای تصمیم | `specs/02_UI_UX/Employee/Workspace.md` |
| مجوز پیاده‌سازی در Odoo | ندارد |

> این نمونه صرفاً برای مشاهده و اعتبارسنجی بصری تصمیمات UX تهیه شده است. وجود آن در مخزن به معنی نهایی‌شدن معماری، مدل داده، API، Migration یا مجوز تغییر ماژول‌های Odoo نیست.

## دامنه تغییرات نسخه ۴

- تبدیل صفحه اصلی از داشبورد اطلاعاتی به Workspace عملیاتی
- برجسته‌سازی «ثبت سریع فعالیت»
- انتخاب فعالیت استاندارد، مدت و توضیح کوتاه
- امکان ثبت فعالیت جدید بدون توقف گزارش روزانه
- ایجاد درخواست استانداردسازی برای فعالیت جدید
- نمایش تدریجی فعالیت‌های امروز
- نمایش مجموع زمان ثبت‌شده
- بازطراحی برنامه امروز و کارهای نیازمند اقدام
- خلاصه‌سازی اطلاعات حضور و شیفت
- میانبرهای شخصی و حالت نمایشی شخصی‌سازی
- حفظ RTL، زبان فارسی و Responsive Design

## محدودیت‌های نمونه

- داده‌ها نمایشی و درون مرورگر هستند.
- Backend و API واقعی وجود ندارد.
- تغییرات در Odoo یا ماژول‌های عملیاتی اعمال نشده‌اند.
- ACL، Record Rule، Workflow و Approval واقعی اجرا نمی‌شوند.
- شخصی‌سازی و ثبت فعالیت‌ها در این نمونه دائمی نیست.
