# Workspace v7 Changelog

## Added
- پنل تنظیمات ظاهر و خوانایی
- سه مقیاس متن
- تم روشن و تاریک
- چهار Accent Color کنترل‌شده
- تراکم فشرده و راحت
- Sidebar جمع‌شونده با حفظ وضعیت
- Quick Conversations در Topbar
- ویجت گفت‌وگوهای اخیر در میزکار
- Drawer اطلاعات گفت‌وگو

## Changed
- نام «پیام‌ها» به «گفت‌وگوها»
- بازطراحی کامل تجربه گفت‌وگو
- افزایش سراسری خوانایی و اندازه متون
- انتقال گفت‌وگوها به سطح اول منوی روز کاری
- یکپارچه‌سازی زبان بصری گفتگو با Workspace

## Preserved
- ساختار اصلی تقویم
- تفکیک کار، تقویم و گزارش فعالیت
- ساختار Action Center

## Known limitations
- داده‌ها نمایشی هستند.
- تنظیمات فعلاً در Local Storage ذخیره می‌شوند.
- جست‌وجو و ارسال فایل نمایشی‌اند.
- اتصال Odoo Mail/Bus، ACL و Record Rule در این Prototype انجام نشده است.
