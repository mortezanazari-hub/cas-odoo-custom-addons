# گزارش تغییرات کنترل‌شده v6 → v7

## اصل پایه
نسخه v7 از روی ZIP اصلی v6 کپی شده است. هیچ فایل، Route، Renderer، نقش یا صفحه‌ای حذف نشده است.

## فایل‌های تغییرکرده
- `index.html`: فقط کنترل Sidebar، Quick Conversations، Appearance و عنوان نسخه
- `assets/data.js`: فقط عنوان نمایشی Route گفتگوها
- `assets/app.js`: تنظیمات ظاهری، Drawer گفتگو، ویجت میزکار و Actionهای مرتبط
- `assets/styles.css`: اصلاح حداقل فونت و Overrideهای v7
- `README_FA.md`: توضیح دامنه نسخه

## فایل‌های بدون تغییر محتوایی
تمام فایل‌های مستندات v6، قرارداد Odoo، Route Catalog، Access Resolver و تصمیم‌های محصول حفظ شده‌اند.

## تضمین‌های طراحی
- میزکار و میز عملیات v6 جایگزین نشده‌اند.
- تقویم v6 بازنویسی نشده است.
- Route فنی `messages` برای سازگاری حفظ شده است.
- گفتگوها و داده‌های قبلی حذف نشده‌اند.
- Collapse در حالت بسته نیز کنترل بازکردن دارد.
