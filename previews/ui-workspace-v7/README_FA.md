# CAS UI Workspace v7

نسخه هفتم Prototype فضای کار سازمانی CAS، بر پایه بازخورد مستقیم روی نسخه ۶.

## تغییرات اصلی

- حذف اندازه‌های بسیار ریز و تعریف مقیاس Typography مرکزی
- حداقل اندازه متن کاربردی: ۱۲ پیکسل
- سه حالت اندازه متن: فشرده، استاندارد و بزرگ
- تغییر عنوان «پیام‌ها» به «گفت‌وگوها»
- بازطراحی صفحه گفت‌وگوها به‌عنوان بخشی یکپارچه از CAS، نه یک پیام‌رسان وصله‌شده
- قرارگیری گفت‌وگوها در سطح اول منوی روز کاری
- اضافه‌شدن دسترسی سریع گفت‌وگو در نوار بالا
- اضافه‌شدن ویجت «گفت‌وگوهای اخیر» در میزکار
- اضافه‌شدن تنظیمات رنگ، تم، اندازه متن، تراکم و وضعیت منوی کناری
- اضافه‌شدن قابلیت Collapse/Expand برای Sidebar
- حفظ تنظیمات کاربر در Local Storage
- حفظ ساختار تقویم و اصلاح خوانایی آن

## اجرا

فایل `index.html` را مستقیماً در مرورگر باز کنید.

## وضعیت

این نسخه Prototype است و داده‌ها نمایشی هستند. هیچ تغییری در Odoo Core یا Repository اصلی انجام نشده است.
