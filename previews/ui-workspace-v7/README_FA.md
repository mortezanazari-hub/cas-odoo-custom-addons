# CAS UI Workspace v7

نسخه هفتم Prototype فضای کار سازمانی CAS، بر پایه نسخه ۶ و بازخوردهای مرحله‌به‌مرحله ارزیابی UX.

## اصل نسخه

نسخه ۷ بازنویسی مستقل سامانه نیست. تمام Routeها، نقش‌ها، صفحات و منطق نمایشی نسخه ۶ حفظ شده‌اند و تغییرات به‌صورت Patch کنترل‌شده روی همان ساختار اعمال شده‌اند.

## مهم‌ترین تغییرات

- افزایش حداقل اندازه متن کاربردی به ۱۲ پیکسل
- سه مقیاس خوانایی: فشرده، استاندارد و بزرگ
- تغییر عنوان نمایشی «پیام‌ها» به «گفت‌وگوها» با حفظ Route فنی `messages`
- دسترسی سریع گفتگو از Topbar و میزکار
- انتقال اطلاعات گفتگو به Drawer درخواستی
- Sidebar با Collapse/Expand پایدار و حفظ وضعیت کاربر
- Accent Color سراسری و Dark Mode
- ویجت‌های قابل جابه‌جایی با ذخیره ترتیب
- ارتفاع ثابت ویجت‌ها و اسکرول داخلی Body
- ردیف‌های خالی غیرفعال برای حفظ ظرفیت بصری فهرست‌ها
- تقویم روز، هفته و ماه داخل میزکار
- نمای ماهانه شمسی با نام ماه، سال، روزهای هفته و ناوبری ماه قبل/بعد
- ادغام میانبرهای تکراری در نوار عملیات میزکار
- Hero منظم سه‌ردیفه شامل خوش‌آمدگویی، وضعیت‌های روزانه و جست‌وجوی سراسری

## وضعیت Prototype

- داده‌ها نمایشی هستند.
- تعامل‌ها برای ارزیابی UX پیاده‌سازی شده‌اند.
- اتصال نهایی به مدل‌ها، ACL، Record Rule، Mail/Bus و Calendar در مرحله پیاده‌سازی Odoo انجام می‌شود.
- هیچ تغییری در Odoo Core در نظر گرفته نشده است.
