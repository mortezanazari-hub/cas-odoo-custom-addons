# UI Review Cycle 9 — Iteration 9

## تغییرات پذیرفته‌شده
1. کنترل تنظیمات میزکار فقط به صورت آیکون چرخ‌دنده در Topbar و فقط در Route `home` نمایش داده می‌شود.
2. کنترل متنی تنظیمات از Hero آبی میزکار حذف شد.
3. گروه ناوبری «فرم‌ها و فرایندها» برای کاربران دارای capability `form.design` ایجاد شد.
4. زیرمنوهای این گروه: `forms-admin` و `form-builder`.
5. صفحه `forms-admin` دارای CTA مستقیم برای ورود به `form-builder` است.

## مسیرهای بازبینی
- میزکار ← نمای کلی میزکار ← آیکون چرخ‌دنده در هدر
- تغییر کاربر آزمایشی به «نگار یوسفی — طراح فرم» ← ابزارهای تخصصی ← فرم‌ها و فرایندها ← طراح دیداری فرم
