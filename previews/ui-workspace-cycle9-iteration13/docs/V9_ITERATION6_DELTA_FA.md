# UI Review Cycle 9 — Iteration 6

## اصلاح ثبت فعالیت خارج از فهرست

- گزینه «فعالیت در فهرست نیست» دیگر صرفاً یک مقدار داخل Select نیست.
- انتخاب این گزینه فوراً فرم ثبت فعالیت جدید را باز می‌کند.
- یک دکمه مستقل و قابل مشاهده برای همین اقدام کنار فرم ثبت سریع اضافه شد.
- فرم ثبت شامل عنوان، دسته‌بندی، مدت و توضیح است.
- پس از ثبت، فعالیت بلافاصله وارد گزارش روز می‌شود و با برچسب «عنوان پیشنهادی» نمایش داده می‌شود.
- پیشنهاد استانداردسازی مانع ارسال گزارش روزانه نیست.
