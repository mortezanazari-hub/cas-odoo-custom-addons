# UI Review Cycle 9 — Iteration 12 (Final Fixes)

## محدوده
اصلاح نهایی ویجت ثبت سریع گزارش کار پیش از Freeze شدن Cycle 9.

## تغییرات پذیرفته‌شده
1. کنترل زمان دلخواه فقط در حالت انتخاب «زمان دلخواه» ظاهر می‌شود.
2. ساختار Layout کنترل مدت از یک Grid ثابت دو ستونه به Layout شرطی تغییر کرد تا هم‌پوشانی ایجاد نشود.
3. دکمه مستقل «فعالیت در فهرست نیست» حذف شد. مسیر ثبت عنوان جدید صرفاً از گزینه موجود در کنترل انتخاب فعالیت اجرا می‌شود.
4. کنترل فعالیت از Select ساده به ورودی جست‌وجوشونده با پیشنهادهای معتبر تبدیل شد.
5. ثبت مقدار آزاد و نامعتبر به‌عنوان فعالیت استاندارد مسدود شده است.
