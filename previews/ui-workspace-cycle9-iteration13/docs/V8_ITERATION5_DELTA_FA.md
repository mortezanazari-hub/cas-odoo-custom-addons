# تغییرات نسخه ۸ — Iteration 5

## فهرست گفت‌وگوها

- علت ارتفاع غیرعادی ردیف‌ها، کشیده‌شدن سطرهای Grid برای پرکردن کل ارتفاع پنل بود.
- فهرست گفت‌وگوها به ستون Flex با ردیف‌های `flex: 0 0 auto` تبدیل شد.
- ارتفاع ردیف‌ها به حالت فشرده ۵۸ پیکسل کاهش یافت.
- اندازه آواتار، عنوان، پیش‌نمایش، زمان و نشان خوانده‌نشده متناسب با ردیف فشرده تنظیم شد.
- فضای خالی پنل دیگر بین ردیف‌ها توزیع نمی‌شود و بعد از آخرین گفت‌وگو باقی می‌ماند.

## تقویم ماهانه میزکار

- جهت نمادهای ناوبری ماه در رابط RTL اصلاح شد.
- دکمه «ماه قبل» اکنون نماد `‹` و دکمه «ماه بعد» نماد `›` دارد.
- `aria-label` و `title` دکمه‌ها برای جلوگیری از ابهام حفظ و تکمیل شدند.

## کنترل پذیرش

- هر گفت‌وگو فقط به اندازه محتوای خود ارتفاع می‌گیرد.
- پنج گفت‌وگو نباید کل ارتفاع پنل را بین خود تقسیم کنند.
- کلیک ماه قبل مقدار offset را کم و کلیک ماه بعد آن را زیاد می‌کند.
- جهت بصری فلش‌ها با معنای فارسی/RTL دکمه‌ها هماهنگ است.
