# تغییرات بازنگری دوم نسخه ۷

این بازنگری بر پایه نسخه v7 اصلاح‌شده و بدون حذف Routeها یا صفحات v6 انجام شده است.

## میزکار
- شبکه ویجت‌های فشرده و قابل جابه‌جایی
- ذخیره ترتیب ویجت‌ها در Local Storage
- انتقال گفت‌وگوها به بخش قابل مشاهده میزکار
- ادغام دو ویجت بزرگ فعالیت
- حذف ویجت تکراری میانبرها و ادغام آن با نوار عملیات

## تقویم
- نمای روز، هفته و ماه در خود ویجت میزکار
- تقویم کامل نسخه ۶ همچنان حفظ شده است

## ظاهر
- Dark Mode
- Accent Color سراسری
- اصلاح کنترل Collapse
- آیکن اختصاصی دو حباب برای گفتگو
