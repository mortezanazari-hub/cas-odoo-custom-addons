# UI Review Cycle 9 — Iteration 11

## تغییرات

- کارت بزرگ «مرکز فرمان شخصی» به فهرست اجزای قابل نمایش میزکار افزوده شد.
- کاربر می‌تواند آن را فقط از آیکون تنظیمات میزکار در هدر مخفی یا دوباره فعال کند.
- هیچ پیام عمومی درباره وجود بخش مخفی در میزکار نمایش داده نمی‌شود.
- بخش «میانبرهای سریع» در همان پنل تنظیمات امکان افزودن، حذف و تغییر ترتیب میانبرها را فراهم می‌کند.
- تصمیم فعلی درباره ظاهر نشان حضور بدون تغییر باقی ماند.

## مسیر بازبینی

`میزکار → آیکون چرخ‌دنده هدر → ویجت‌های قابل نمایش / میانبرهای سریع`
