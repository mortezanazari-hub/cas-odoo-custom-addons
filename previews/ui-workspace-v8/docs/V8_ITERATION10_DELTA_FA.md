# نسخه ۸ — Iteration 10

## هدف
افزایش داده آزمایشی گفت‌وگوی فعال برای ارزیابی عملی Scroll داخلی پنل پیام‌ها.

## تغییر
- تعداد پیام‌های نمونه از ۴ پیام به ۱۶ پیام افزایش یافت.
- ارتفاع Route گفت‌وگو، Header و Composer تغییری نکرد.
- پیام‌ها فقط داخل `.chat-body` اسکرول می‌شوند.
- فهرست گفت‌وگوها همچنان Scroll مستقل خود را دارد.
- این تغییر صرفاً داده نمایشی Prototype است و قرارداد داده Production را تغییر نمی‌دهد.

## معیار پذیرش
- با بازشدن صفحه گفت‌وگو، پیام‌های اضافی باعث افزایش ارتفاع کل Route نشوند.
- Scrollbar و Auto-scroll دکمه وسط موس فقط در پنل پیام‌ها عمل کنند.
- Composer در پایین ثابت باقی بماند.
- Header گفت‌وگو ثابت باقی بماند.
