# نسخه ۸ — Iteration 6

## صفحه نیازمند اقدام

- فیلتر منبع از Dropdown به چیپ‌های تک‌انتخابی تبدیل شد.
- منابع: همه منابع، حضور و غیاب، فرم و فرایند، مکاتبات.
- فقط یک منبع در هر لحظه فعال است و انتخاب جدید جایگزین قبلی می‌شود.
- فیلتر زمانی مستقل و تک‌انتخابی باقی مانده است.
- مرتب‌سازی همچنان Dropdown است تا با مفهوم فیلتر اشتباه نشود.
- تعداد هر منبع داخل چیپ نمایش داده می‌شود.
- در موبایل چیپ‌ها اسکرول افقی دارند.
- نتیجه فهرست واقعاً بر اساس چیپ فعال فیلتر می‌شود.
