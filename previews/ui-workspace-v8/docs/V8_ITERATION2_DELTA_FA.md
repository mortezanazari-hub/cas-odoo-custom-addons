# تغییرات نسخه ۸ — Iteration 2

## گفتگوها
- بسته‌شدن Context Menu با کلیک بیرون
- حذف اسکرول صفحه و محدودشدن اسکرول به فهرست پیام‌ها
- ثابت‌ماندن Composer در پایین پنجره
- فشرده‌سازی آیتم‌های فهرست گفتگو
- فعال‌شدن انتخابگر Emoji برای واکنش پیام و Composer

## تقویم
- حذف الگوی نمایش همه کارکنان
- جست‌وجوی نام، سمت و واحد
- فیلتر دپارتمان
- فیلتر «فقط زیرمجموعه‌های من»
- تعیین روش ارسال برای هر شرکت‌کننده
- محدودشدن «وظیفه» به زیرمجموعه مجاز
- امکان دعوت‌نامه برای افراد خارج از زیرمجموعه

## قاعده امنیتی
محدودیت تعریف وظیفه باید در Backend نیز براساس ساختار سازمانی، انتساب معتبر، شرکت جاری، ACL و Record Rule کنترل شود؛ غیرفعال‌کردن گزینه در UI به‌تنهایی امنیت نیست.
