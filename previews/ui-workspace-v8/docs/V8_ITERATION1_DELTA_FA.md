# تغییرات نسخه ۸ — Iteration 1

## کارهای من
- دسته‌های سیستمی قفل و غیرقابل حذف هستند.
- دسته‌های شخصی قابل افزودن، ویرایش، حذف و فیلتر هستند.
- حذف دسته شخصی، کارهای آن را به دسته «شخصی» منتقل می‌کند.

## تقویم
- انتخاب واقعی شرکت‌کنندگان به فرم رویداد افزوده شد.
- سه شیوه ارسال وجود دارد: دعوت‌نامه تقویمی، وظیفه کارتابل، یا هر دو.
- توضیحات رویداد از فهرست شرکت‌کنندگان جدا شد.

## گفت‌وگوها
- آیکن نامه با نماد گفت‌وگو جایگزین شد.
- سربرگ معرفی صفحه و جست‌وجوی تکراری حذف شد.
- دکمه گفت‌وگوی جدید به‌صورت شناور روی لیست قرار گرفت.
- آواتارها دایره‌ای و هم‌اندازه شدند.
- اسکرول فقط در لیست گفتگوها و بدنه پیام‌ها انجام می‌شود؛ Composer ثابت است.
- Reply، Forward، حذف پیام خود کاربر، Pin پیام، Pin گفتگو، Mute و Archive در Context Menu اختصاصی افزوده شد.
- Emoji Reaction و Reply Preview افزوده شد.
- این Prototype قرارداد استفاده از قابلیت‌های استاندارد Odoo Mail/Discuss/Bus را نمایش می‌دهد و مالکیت پیام را به Workspace منتقل نمی‌کند.
