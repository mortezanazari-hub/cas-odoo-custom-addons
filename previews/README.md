# CAS UI Prototypes

این پوشه برای نگهداری پیش‌نمایش‌های رابط کاربری CAS استفاده می‌شود.

> تمام فایل‌های این مسیر صرفاً **پیش‌نمایش غیرقطعی** هستند و به‌تنهایی مبنای پیاده‌سازی در Odoo محسوب نمی‌شوند. تصمیمات رسمی محصول در مسیر `specs/` نگهداری می‌شوند.

## نسخه‌ها

- `workspace-v4/` — پیش‌نمایش بازطراحی صفحه میزکار کاربر عادی بر اساس تصمیمات Product Design ثبت‌شده در `specs/02_UI_UX/Employee/Workspace.md`.

## قواعد نسخه‌گذاری

- هر نسخه در پوشه‌ای مستقل نگهداری می‌شود.
- نسخه‌های قبلی حذف یا جایگزین نمی‌شوند.
- هر پوشه باید وضعیت، دامنه تغییر و محدودیت‌های نمونه را در README خود اعلام کند.
- وجود نمونه در این مسیر به معنی تأیید معماری، مدل داده یا آماده‌بودن برای پیاده‌سازی نیست.
