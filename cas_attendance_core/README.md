# CAS Attendance Core

لایه ممیزی‌پذیر ورود و خروج نگهبانی و دستگاه برای Odoo 19.

- نگهداری تغییرناپذیر زمان واقعی رخداد و زمان ثبت در سامانه
- انتساب مستقل رخداد به روز کاری، با پشتیبانی شیفت شب
- تطبیق ساده با قاعده پنج‌دقیقه‌ای و ارجاع اختلاف به سرپرست
- تکمیل ترکیبی منابع با هشدار و جایگزینی نگهبانی هنگام خرابی دستگاه
- مدل رخداد و بازه برای حالت پیشرفته، استراحت، خروج شخصی، مأموریت و مرخصی ساعتی
- ابطال بدون حذف سابقه و جلوگیری از دریافت تکراری رخداد دستگاه

این افزونه عمداً کاردکس، درخواست‌ها و محاسبه اضافه‌کاری را انجام نمی‌دهد؛ آن‌ها لایه بعدی هستند.
