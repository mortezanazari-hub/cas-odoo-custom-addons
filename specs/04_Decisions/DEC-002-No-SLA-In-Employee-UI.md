# DEC-002 — SLA فنی در UI کاربر عادی نمایش داده نمی‌شود

| مشخصه | مقدار |
|---|---|
| وضعیت | `Agreed` |
| نسخه | `v8` |
| تاریخ تثبیت | `2026-07-21` |

## زمینه

نمایش عدد و اصطلاح SLA فنی برای کاربر عادی باعث پیچیدگی و فاصله از اقدام واقعی می‌شود، در حالی که Backend ممکن است همچنان از SLA برای محاسبه اولویت استفاده کند.

## تصمیم

در UI عمومی از واژگان قابل‌فهم استفاده می‌شود:

- فوری
- نزدیک موعد
- عقب‌افتاده
- نیازمند اقدام

SLA فنی، زمان‌بندی و Ruleهای محاسباتی در Domain مالک باقی می‌مانند و در صفحات تخصصی مدیریتی یا Audit در صورت مجوز قابل نمایش‌اند.

## پیامدها

- Workspace فقط Label و Priority Resolve‌شده را نمایش می‌دهد.
- حذف SLA از UI به معنی حذف Deadline یا Rule نیست.
- Search و Notification از همان واژگان UX استفاده می‌کنند.

## گزینه ردشده

نمایش مستقیم اصطلاحات و شمارنده‌های SLA در تمام Cardهای کاربر عادی رد شد.
