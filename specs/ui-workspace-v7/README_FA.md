# مشخصات CAS UI Workspace v7

این پوشه مرجع رسمی مشخصات نسخه نهایی فعلی Workspace است.

## خط مبنا

خط مبنای رسمی این نسخه، بسته `CAS_UI_Prototype_V4` است. نسخه‌های ۵ و ۶ Release رسمی مستقل محسوب نمی‌شوند و صرفاً Iterationهای داخلی طراحی بوده‌اند.

```text
CAS UI Prototype v4 → CAS UI Workspace v7
```

نسخه ۷ جایگزین معماری نسخه ۴ نیست. ساختار نقش‌محور، صفحات تخصصی، Routeها، منطق دسترسی و قراردادهای اتصال به Odoo حفظ شده و تجربه عمومی Workspace روی همان پایه توسعه یافته است.

## مهم‌ترین تغییرات نسبت به نسخه ۴

- توسعه میزکار کاربر عادی به مرکز فرمان شخصی
- Briefing روزانه، کارهای من، نیازمند اقدام و جست‌وجوی سراسری
- Hero سه‌ردیفه و منظم
- حداقل اندازه متن کاربردی ۱۲ پیکسل
- تغییر عنوان نمایشی «پیام‌ها» به «گفت‌وگوها» با حفظ Route فنی `messages`
- دسترسی گفتگو از Sidebar، Topbar و میزکار
- Sidebar با Collapse/Expand پایدار
- Accent Color سراسری و Dark Mode
- Widgetهای قابل Drag با ذخیره ترتیب
- ارتفاع ثابت Widgetها و Scroll داخلی Body
- ردیف‌های خالی غیرفعال برای حفظ ریتم بصری
- تقویم تعاملی روز، هفته و ماه در میزکار
- نمای ماهانه شمسی با عنوان ماه، سال، روزهای هفته و ناوبری
- ادغام میانبرهای تکراری در نوار عملیات
- حفظ صفحات تخصصی نگهبانی، دبیرخانه، حضور، شیفت، فرم، گردش‌کار، اسناد و مدیریت سامانه

## اصول معماری

- هیچ تغییری در Odoo Core انجام نمی‌شود.
- Workspace مالک داده کسب‌وکاری نیست.
- داده‌ها از مدل‌های واقعی Odoo و با رعایت ACL و Record Rule خوانده می‌شوند.
- استفاده از `sudo` برای دورزدن دسترسی مجاز نیست.
- Jalali فقط لایه ورود، نمایش و فیلتر تاریخ است.
- رابط کاملاً فارسی، راست‌چین و مستقل از ظاهر پیش‌فرض Odoo باقی می‌ماند.
