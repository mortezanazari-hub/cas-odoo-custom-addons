# 06 — Change Sets

هر جلسه یا دامنه نهایی‌شده یک Change Set غیراجرایی در این بخش دارد. Change Set تصمیم‌ها، Routeها، Capabilityها و ماژول‌های احتمالی را ثبت می‌کند، اما تا زمان تجمیع ماژولی مجوز تغییر کد نیست.
